USTORE.js demo

One less thing to complain about not being able to use in IE...

USTORE.js is a cross browser solution for sessionStorage and localStorage. This file serves as a demo of USTORE.js' core functionality. To get under the hood, check out the init.js file to see how this page interacts with USTORE.js. To really figure things out, check out the actual USTORE.js library. Don't forget to open a few different tabs and/or windows to test out session storage!

Set Values

Enter a key name, then enter it's value, then hit save. Your key = value pair will be stored in the browser's local storage. To store it in the browser window's session storage, click the checkbox next to save.

Get Values

Enter a key you stored in the above form to get it's value. If you stored it locally, close your browser, come back to this page, and then try and get it. Still there? Thought so. If it's stored in the session, don't close your browser, and click the checkbox to search just in the session.

Delete Values

Delete a value by it's key name. To delete a value from the session and not the local storage, mark the checkbox.

Clear Values

Clear all the values in both the local & session storages, or just one.
